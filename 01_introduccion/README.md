# Introducción

## ¿Qué es OmegaT?

OmegaT es una herramienta de traducción asistida por ordenador, desarrollada en Java, y por tanto multiplataforma, siguiendo un modelo de desarrollo basado en código abierto (licencia pública general GNU).
